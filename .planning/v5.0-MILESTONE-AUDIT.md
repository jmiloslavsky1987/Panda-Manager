## Broken E2E Flow

### Flow: Ingestion Approval → Dashboard Refresh

**Steps 1–5:** ✓ Complete — Upload, extract, review, approve, entities written to DB

**Step 6:** ✗ Missing — `IngestionModal.handleApprove()` calls `setStage('done')` but never fires:
```typescript
window.dispatchEvent(new CustomEvent('metrics:invalidate'))
router.refresh()
```

**Impact:** After approving an ingestion, the Overview tab shows stale entity counts until the user manually refreshes the page.

**Fix location:** `bigpanda-app/components/IngestionModal.tsx` — add event dispatch after line ~352 where stage is set to 'done'.
