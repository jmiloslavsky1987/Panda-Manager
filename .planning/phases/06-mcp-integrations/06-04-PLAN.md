- `export async function POST(request: Request)`

- Makes a lightweight test: attempt to fetch the MCP server URL with a HEAD or GET request (timeout 5s) with `Authorization: Bearer {apiKey}` header
- Returns `{ ok: true }` on 2xx response
- Returns `{ ok: false, error: string }` on non-2xx or network error
- Do NOT call Anthropic SDK here — just check reachability of the MCP server URL
- Use Node.js `fetch` with `AbortSignal.timeout(5000)` for the timeout
