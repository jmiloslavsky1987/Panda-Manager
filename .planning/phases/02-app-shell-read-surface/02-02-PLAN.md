Implement these exported async functions:

```typescript
// Returns all active projects with computed health score
export async function getActiveProjects(): Promise<ProjectWithHealth[]>
```
- Query `projects` WHERE `status = 'active'`
- For each project, compute health score:
  - Count overdue actions: actions WHERE project_id = p.id AND status IN ('open','in_progress') AND due < today (exclude 'TBD'/'N/A' strings)
  - Count stalled milestones: milestones WHERE project_id = p.id AND status != 'completed' AND created_at < now() - interval '14 days' (use created_at timestamp — milestones table has no last_updated column; created_at is used as a proxy: a milestone created 14+ days ago that's still incomplete is a stall signal)
  - Count unresolved high-severity risks: risks WHERE project_id = p.id AND severity IN ('high','critical') AND status != 'resolved' AND status != 'closed'
  - RAG score formula: score = overdueActions + stalledMilestones + highRisks; if score >= 2 → 'red', if score === 1 → 'yellow', if score === 0 → 'green'
- Return type `ProjectWithHealth`: project fields + `{ health: 'green' | 'yellow' | 'red', overdueActions: number, highRisks: number, stalledMilestones: number }`
