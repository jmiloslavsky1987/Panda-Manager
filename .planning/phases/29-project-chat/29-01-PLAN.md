    chat-context-builder.ts:
    - buildChatContext(projectId: number): Promise&lt;string&gt;
    - Returns markdown string with sections: Project info, Open Actions (with [external_id] prefixes), Open Risks, Milestones, Key Stakeholders, Workstreams, Key Decisions, Recent Engagement History (last 20)
    - Open actions filtered: status !== 'completed' AND status !== 'cancelled'
    - Open risks filtered: status !== 'closed'
    - Each action line: `- [${external_id}] ${description} | Owner: ${owner ?? 'TBD'} | Due: ${due ?? 'TBD'} | Status: ${status}`
    - Each risk line: `- [${external_id}] ${description} | Severity: ${severity ?? 'N/A'} | Mitigation: ${mitigation ?? 'None'}`
    - All queries delegated to getProjectById(projectId) + getWorkspaceData(projectId) — never queries without projectId filter

    - Returns result.toUIMessageStreamResponse()
    - System prompt contains ALL 5 anti-hallucination constraints listed in interfaces above
