<objective>
Enhance the discovery scanner to identify items that likely duplicate existing project data, and surface that signal clearly in the review queue.

Purpose: After source adapters sweep data and Claude extracts structured items, a second signal is needed: "does this item already exist in the project?" This prevents the queue from being flooded with things already captured, and makes review faster and more accurate. Claude already classifies fields correctly — this adds a redundancy flag on top.

Output: Scanner fetches compact project context, passes it to Claude analysis, Claude flags likely duplicates, queue renders them separately.
</objective>

**A. Update scan route** `app/api/discovery/scan/route.ts` — fetch existing project items and build compact summary before calling `runDiscoveryScan`:

After the project lookup (step 1), add:

```typescript
// Build compact project context for dedup analysis
// Fetch existing actions, risks, stakeholders, decisions (limit to 50 each to stay within token budget)
import { actions, risks, stakeholders } from '@/db/schema';

const [existingActions, existingRisks, existingStakeholders] = await Promise.all([
  db.select({ id: actions.id, title: actions.title, status: actions.status })
    .from(actions).where(eq(actions.project_id, projectId)).limit(50),
  db.select({ id: risks.id, title: risks.title, status: risks.status })
    .from(risks).where(eq(risks.project_id, projectId)).limit(50),
  db.select({ id: stakeholders.id, name: stakeholders.name, role: stakeholders.role })
    .from(stakeholders).where(eq(stakeholders.project_id, projectId)).limit(50),
]);

const existingProjectSummary = [
  existingActions.length > 0
    ? `Actions:\n${existingActions.map(a => `- [${a.status}] ${a.title}`).join('\n')}`
    : null,
  existingRisks.length > 0
    ? `Risks:\n${existingRisks.map(r => `- [${r.status}] ${r.title}`).join('\n')}`
    : null,
  existingStakeholders.length > 0
    ? `Stakeholders:\n${existingStakeholders.map(s => `- ${s.name} (${s.role ?? 'unknown role'})`).join('\n')}`
    : null,
].filter(Boolean).join('\n\n');
```
