## Pattern from Research

Per 074-RESEARCH.md Pattern 3 (Standalone Output Docker Build):
- Set `output: "standalone"` in next.config.ts
- Enables file tracing for minimal .next/standalone bundle
- Required for production Docker deployments

Per 074-RESEARCH.md Anti-Patterns:
- "Hardcoded localhost fallbacks in production code: Use environment variables with NO defaults for production-required config (DATABASE_URL, REDIS_URL)"
- Fallbacks mask configuration errors and cause silent failures in production

```typescript
const url = process.env.REDIS_URL ?? 'redis://localhost:6379';

const url = process.env.REDIS_URL!;

}
```

Why: Invite emails with wrong callback URLs are worse than failing fast. Production must have BETTER_AUTH_URL set to actual domain.

With:
