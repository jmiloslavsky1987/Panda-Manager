    - Fetch events from Google Calendar API for the specified week (weekStart to weekStart+7d)
    - Filter: skip all-day events with no time component, skip events shorter than 5 minutes
    - For each event, compute project match:
      1. Fetch all projects with stakeholders: SELECT projects.id, projects.customer, stakeholders.email FROM stakeholders JOIN projects ON ...
      2. For each event.attendees, check emails against stakeholder emails
      3. matched_project_id = project with most attendee matches (null if no match)
    - Return array of:
      ```typescript
      {
        event_id: string;          // Google Calendar event ID
        summary: string;           // event title → becomes description
        date: string;              // event.start.dateTime split to YYYY-MM-DD (TTADV-14: use event date)

        matched_project_name: string | null;
        match_confidence: 'high' | 'low' | 'none';  // high: 2+ attendee matches, low: 1 match
      ```
