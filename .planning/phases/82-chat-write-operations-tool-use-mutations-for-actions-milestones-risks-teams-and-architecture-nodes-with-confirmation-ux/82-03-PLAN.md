Operation type detection from part.type:
```typescript
// part.type format: 'tool-{toolName}'
// toolName starts with 'create_' → create operation (green)
// toolName starts with 'update_' → update operation (blue/interactive)
// toolName starts with 'delete_' → delete operation (red)
const toolName = part.type.replace('tool-', '')
const opType = toolName.startsWith('create_') ? 'create'
  : toolName.startsWith('update_') ? 'update'
  : 'delete'
```
