    **Verification 1 — Global toggle (SKILL-03a):**

    10. Toggle ON (as admin). Open Skills tab. Click Edit on any skill (e.g., "meeting-summary").
    11. Modal opens. Verify:

        - Markdown toolbar visible (Bold, Italic, Code, Heading buttons)

    **Verification 4 — Resize handle (SKILL-03b):**
    12. Drag the resize handle at the bottom of the editor area to make it taller and shorter.
    13. Confirm the editor area resizes vertically as expected.

    **Verification 5 — Full-screen (SKILL-03b):**
    14. Click the full-screen toggle. Editor expands to near full-viewport height (90vh+).
    15. Click again to exit full-screen. Editor returns to standard size.

    **Verification 6 — Successful save (SKILL-03b):**
    16. Make a trivial edit to the body (add/remove a space or word).
    17. Click Save. Modal closes. No error shown.
    18. Reopen the edit modal for the same skill. Confirm the edit persists.
    19. Check the skills directory for a `.bak` file for the edited skill.

    **Verification 7 — Validation failure (SKILL-03b):**
    20. Open edit modal. Delete all content from the editor body.
    21. Click Save. Inline error appears ("Prompt body does not conform to Design Standard" or similar).
    22. Modal stays open. No file was written (verify by checking skill still works).
