**Layout structure:**

The swimlane is a table-like layout where:
- Each row = one workstream (plus one "Unassigned" row at the bottom)
- Each row has: a left header cell (workstream name + progress bar) and a content area with task cards grouped by status

For status columns within each lane, use 4 mini-columns: To Do / In Progress / Blocked / Done.

```tsx
// Group tasks by workstream_id
const tasksByWorkstream = new Map<number | null, Task[]>()
for (const task of tasks) {
  const key = task.workstream_id
  if (!tasksByWorkstream.has(key)) tasksByWorkstream.set(key, [])
  tasksByWorkstream.get(key)!.push(task)
}

// Lanes: workstreams in DB order + Unassigned lane if any tasks have no workstream_id
const lanes = [
  ...workstreams,
  { id: null, name: 'Unassigned', percent_complete: null, track: null, project_id: projectId } as typeof workstreams[number] & { id: null },
].filter(ws => ws.id === null || (tasksByWorkstream.get(ws.id)?.length ?? 0) > 0)
```

For each lane row:
- Left header (fixed width ~200px, vertically centered):
  - Workstream name in bold
  - Track badge (ADR/Biggy) if ws.track is set
  - Progress bar: `<progress value={ws.percent_complete ?? 0} max={100} className="w-full h-2 mt-1" />`
  - Progress text: `{ws.percent_complete ?? 0}% complete`
- Content area: 4 sub-columns with task cards
  - Status label at top of each sub-column: "To Do" / "In Progress" / "Blocked" / "Done"
  - Task cards for that status filtered within this workstream

Use @dnd-kit DndContext with `id={`swimlane-${ws.id ?? 'unassigned'}`}` per lane to scope drag operations to within each lane. On DragEnd: PATCH `/api/tasks/${taskId}` with `{ status: newStatus }`.

Optimistic state: `const [optimisticTasks, setOptimisticTasks] = useState(tasks)`. On drag start: update state immediately. On error: revert to prop value.

```tsx
import { getTasksForProject, getWorkstreamsWithProgress } from '@/lib/queries'
import { SwimlaneView } from '@/components/SwimlaneView'

export default async function SwimlanePage({
  params,
}: {
  params: Promise<{ id: string }>
}) {
  const { id } = await params
  const projectId = parseInt(id)

  const [tasks, workstreams] = await Promise.all([
    getTasksForProject(projectId),
    getWorkstreamsWithProgress(projectId),
  ])

  return (
    <div className="p-4">
      <div className="flex items-center justify-between mb-4">
        <h2 className="text-lg font-semibold">Team Swimlane</h2>
        <p className="text-xs text-zinc-400">{workstreams.length} workstreams · {tasks.length} tasks</p>
      </div>
      {tasks.length === 0 ? (
        <p className="text-zinc-500 text-sm">No tasks yet. Create tasks in the Phase Board or Task Board first.</p>
      ) : (
        <SwimlaneView tasks={tasks} workstreams={workstreams} projectId={projectId} />
      )}
    </div>
  )
}
```
