**Step 2: Add helper function `computeTrend()`:**
```typescript
function computeTrend(current: number, prior: number): 'up' | 'flat' | 'down' {
  const diff = current - prior;
  if (Math.abs(diff) <= 1) return 'flat';
  return diff > 0 ? 'up' : 'down';
}
```
